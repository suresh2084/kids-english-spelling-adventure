# Kids English Spelling Adventure

A fun, gamified English spelling app for children aged 4–8, built with React + Vite.

## Features
- 5 progressive levels (3-letter to 7+ letter words)
- 250+ words across all levels
- Text-to-Speech pronunciation (Indian English accent preferred)
- Hearts lives system, coins, badges & rewards shop
- Daily challenge & parent dashboard
- Accessibility: dyslexia font, high contrast, dark mode
- All progress saved to localStorage — no account needed

## Tech Stack
React 18 + TypeScript · Vite · Tailwind CSS · shadcn/ui · Framer Motion · Wouter · Web Speech API

## Run locally
```bash
npm install
npm run dev
```
